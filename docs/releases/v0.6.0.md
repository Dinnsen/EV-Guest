## EV Guest v0.6.0

This release keeps the stable entity model from v0.4.0/v0.5.1 and adds optional charger status awareness, language and country selection in configuration, and a new `Charge Now` diagnostic binary sensor for automations.

### Added
- Optional **Charger Status Entity** in configuration and options
- **Language** selection in configuration: English and Dansk
- **Country** selection in configuration: Denmark
- Extensible country/provider structure for future number-plate lookup providers
- New diagnostic binary sensor: **Charge Now**
- Updated translations for the new configuration fields
- Updated logo assets in all required brand formats

### Changed
- **Charger Switch Entity** is now optional and can be left blank
- README updated as integration documentation, not release notes
- HACS README logo now uses a direct raw image URL so it renders inside Home Assistant/HACS more reliably
- Config entry migration updated to keep existing installations fully migratable

### Compatibility
- Existing entity names and unique IDs are preserved
- Existing installs migrate forward to the new config version without renaming the old entities
- `Charge Now` is a new additional entity; existing entities are untouched

### Notes
- The built-in dummy charger is **not** part of the integration package in v0.6.0
- For testing, use Home Assistant helpers such as `input_boolean` plus a template switch
