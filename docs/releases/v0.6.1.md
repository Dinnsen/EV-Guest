## EV Guest v0.6.1

This release fixes the v0.6.0 config-flow crash and keeps the stable entity model from v0.4.0/v0.5.1.

### Fixed
- Fixed the config flow 500 error caused by optional entity selectors
- `Charger Switch Entity` can remain blank without breaking the UI
- `Charger Status Entity` can remain blank without breaking the UI

### Changed
- Version bumped to `0.6.1`
- README updated to stay as integration documentation
- README logo now uses a direct raw GitHub image URL for better rendering in HACS/Home Assistant
- Added labels/translations for the new configuration fields

### Compatibility
- Existing entity names and IDs are unchanged
- Existing installs migrate forward without renaming entities
- This release is intended as a safe follow-up to v0.6.0
